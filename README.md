# land-use

Комплекс программ для вычислительного моделирования пространственного распределения примесей.  
Используются Python, GeoPandas, scikit-learn.

## Структура проекта

- `lu_main.ipynb` — ноутбук для запуска пайплайна  
- `lu_info.py` — справочные данные и функции вклада  
- `lu_lib.py` — библиотека функций для геометрических и численных операций на исследуемой территории
- `lu_ml.py` — модели Land Use Buffers/Rings
- `segmentation.py` — модель Land Use Segmentation
- `robustness.py` — оценка устойчивости моделей Land Use Buffers/Rings
- `performance.py` — метрики качества моделей, пермутационный подход, диаграмма Тейлора  
- `representativeness.py` — оценка репрезентативности точек наблюдений
- `lu_env.yml` — файл для создания Conda-среды
