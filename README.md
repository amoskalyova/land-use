# land-use

Пайплайн для вычислительного моделирования пространственного распределения примесей.  
Используются Python, GeoPandas, scikit-learn.

## Структура проекта

- `LU_main.ipynb` — ноутбук для запуска пайплайна  
- `LU_info.py` — справочные данные и функции  
- `PyLIB_LU.py` — вспомогательные функции для обработки данных  
- `performance.py` — метрики качества моделей, пермутационный подход, диаграмма Тейлора  
- `lu_env.yml` — файл для создания Conda-среды
